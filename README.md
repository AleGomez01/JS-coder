# 👜 Felicia - Simulador de E-commerce

Proyecto desarrollado como entrega final del curso de JavaScript.

## 🚀 Funcionalidades

* Carga de productos desde archivo JSON mediante fetch
* Renderizado dinámico en el DOM
* Carrito de compras interactivo
* Suma de cantidades por producto
* Eliminación de productos del carrito
* Cálculo automático del total
* Persistencia de datos con localStorage
* Finalización de compra con alerta personalizada

## 🛠️ Tecnologías utilizadas

* HTML
* CSS
* JavaScript
* SweetAlert2 (librería externa)

## 📦 Estructura del proyecto

* `/pages` → páginas HTML
* `/js` → lógica en JavaScript
* `/data` → archivo JSON con productos

## 💡 Objetivo

Simular el flujo completo de compra de un e-commerce, aplicando conceptos de asincronía, manipulación del DOM y almacenamiento local.
